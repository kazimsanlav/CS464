run as: python main.py path_to_dataset
path_to_dataset is the path where test and training data is
example:
(C:\Program Files\Anaconda3) C:\Users\kazIm\Desktop\Python\PythonClassifierApplication1>python main.py C:\Users\kazIm\Desktop\MLhw\Data

You will need some packages installed like numpy or sklearn(These are already available in Anaconda)
Or if you want to run in an IDE like spyder or visual studio, remove sys.args from main and write the path inside main function
